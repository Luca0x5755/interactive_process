# OpenCV 圖片分析

## 如何開始 (Getting Started)

### 繁體中文指南

1.  **初始設定**:
    將 `ia-interactive-process-zh-TW.md` 檔案上傳至您的語言模型。

    接著，發送以下提示詞以啟動領航員：
    > 你是一名互動式流程嚮導。
    你的任務是基於以下用 Mermaid 語法描述的「影像分析流程圖」，一步步引導我完成整個流程。
    這份流程圖是關於如何使用 Python 的 OpenCV 函式庫來處理與分析影像。
